“html이란 무엇인가” (hypertext)


1. Hypertext란?
'HyperText'란 '글'을 의미하는 'Text'에 '~을 뛰어넘은'이라는 의미를 가진 접두사 'hyper'가 붙은 신조어로 '과거의 글의 한계를 초월한 새로운 글'이라는 의미를 갖고 있다. 여기서 말하는 '과거의 글'이란 종이에 기록되어 순차적으로 읽게 되어있는 글을 의 미하며 '새로운 글'은 과거의 글의 '순차성'을 극복한 글을 의미한다. '종이사전'과 '전자사전'의 차이를 생각해보면 이것이 무슨 의미인지 바로 이해할 수 있을 것이다.
즉, Hypertext란 '다른 문서와 “비순차적” 으로 연결될 수 있는 문서'를 의미한다.
 
(텍스트의 정보에 대한 접근인 순차적 접근을 뛰어넘는다는 의미, 일반적인 텍스트의 순차적 접근법을 넘는 “비순차적 접근법”을 표현한


* HTML의 정의
HTML 은 Hyper Text Markup Language 약어로 HyperText (웹 페이지에 서 다른 페이지로 이동할 수 있록 하는 것) 기능을 가진 문서를 만드는 언 어입니다. 다시 말해, 구조를 설계할 때 사용되는 언어로 hyper link 시스템 을 가지고 있으며, 흔히 말하 는 웹 페이지를 위한 마크업 언어라고 할 수 있다.


* HTML의 설명
HTML 은 문단, 제목, 데이터 표를 정의하거나 페이지에 이미지와 동영상을 삽입하 는 등 웹 콘텐츠를 구성하고 의미를 부여하는 데 사용하는 마크업 언어입니다 CSS 는 배경색과 글꼴을 설정하고 콘텐츠를 여러 열에 배치하는 등 HTML 콘텐츠 에 스타일을 적용하는 데 사용하는 스타일 규칙 언어이다.


2. Markup Language란? “Markup”이란 어딘가에 Mark 표기를 해두는 것 이며 문서를 구조적으로 표시하기 위한 것이다. 제목이 어디서부터 어디까지 인지, 본문은 어디서부터 인지 주소는 어디에 있는지 강조할 부분은 어디인지 등 많은 것을 구조적으로 표현할 수 있다.


“Html”에서 말하는 언어라는 것은 다양한 언어들을 통해 사람과 사람 사이의 의사소통을 하기 위한 언어가 아니라 컴퓨터와 의사소통을 하기 위한 언어이므로 Html의 “Makup Language”를 통해 컴퓨터에게 명령하여 무언가 일을 시키는 것이다.

HTML에서 얘기하는 마크업 언어도,C, Jave 등의 언어와 마찬가지로 컴퓨 터와 의사소통하고 명령을 내리기 위한 언어이다. 웹상에 표시하고 싶은 내용등을 HTML 문서로 작성하면 컴퓨터는 웹브라우저를 통해 HTML 문서를 읽어 들이고 모니터에 표현하여 우리에게 보여주는 것이다.
(browser 웹을 둘러 볼 수 있게 하는 응용 프로그램을 말한다.)


Hyper Text를 가능하게 하는 Hyper Link 기능도 마크업에 언어가 담당한다.
HTML 태크를 웹 브라우저가 읽어 들여 Hyper Link 기능을 구현하여 보여준다.
이러한 Tag의 이유는 어떠한 이것이 달려있는 상품에 대한 정보들을 갖고 있기 때문에 HTML에서 Tag 기능도 마찬가지이다.


마크업 언어는 웹 브라우저에게 내 컨텐츠를 어떻게 표현해야 하는지 명령 하는 언어 이며,

함께 일하는 인간들뿐만 안니라 웹 크롤러 라는 데이터를 긁어 수집하는 로봇들에게도 웹 문서 내역 데이터들을 구조적으로 표시하여 정보를 전달하는 언어이다.
