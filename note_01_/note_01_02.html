핵심 요약 본

  • HTML은 크게 head와 body로 구성된다.
  • 태그는 h태그, p태그, link 태그 등 150여개가 존재한다.
  • h태그는 제목을 나타낼 때 사용한다.
  • br, p, div, span 태그는 텍스트를 나눌때 사용한다.
  • link 태그는 현재 파일이 다른 파일과 연결되어 있는 관계 정보를 표현하는데 사용한다.
  • img 태그는 이미지를 불러올떄 사용한다.
  • form 태그는 사용자의 입력을 받는 양식을 만들때 사용한다.
  • button 태그는 버튼을 만들때 사용한다.
  • 리스트 태그(ol,ul)는 항목을 나열할 때 사용한다.
